# Arma-3-wedge-Charge

wedge charge that sticks to surfaces, demolishes walls, and opens doors, with minimal collateral damage.

Class names

`AMP_wedge_Charge_Mag`

Works on doors locked via

```
_house setVariable ["bis_disabled_Door_1", 1, true];
```
